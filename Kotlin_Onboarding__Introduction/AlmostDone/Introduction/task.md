### Project description

The project of this lesson is **Console photoshop**.
The goal of this project is to create a simple Photoshop console app 
that can apply different filters to character images.

### Lesson topics

- multi-row strings;
- when expressions;
- basic handling of errors;
- string builder;
- null safety (`!!` and `?:`).
